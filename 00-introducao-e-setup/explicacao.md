# 🛠️ Preparando o Ambiente Python (Windows)

Bem-vindo(a)! 👋
Antes de começar a programar, vamos preparar o seu computador.

Pense assim:

🐍 Python → é a linguagem que vamos usar
🧑‍💻 VS Code → é onde você vai escrever seus códigos

Relaxa, vamos passo a passo.

✅ Passo 1: Instalando o Python

O Python é essencial. Sem ele, nada roda.

🔽 Download
Acesse: https://www.python.org/downloads/
Clique em Download Python (versão mais recente)
⚙️ Instalação
Abra o arquivo baixado
⚠️ MUITO IMPORTANTE:

👉 Marque a opção:

✔ Add Python to PATH
Depois clique em:
Install Now

⏳ Aguarde a instalação finalizar

🧪 Testar se funcionou
Aperte Win + R
Digite: cmd
No terminal, digite:
python --version

✔ Se aparecer algo como:

Python 3.x.x

Tudo certo! 🎉

🧑‍💻 Instalando o VS Code (Seu Editor de Código)

Agora precisamos de um lugar para escrever nossos códigos.

O VS Code (Visual Studio Code) é leve, gratuito e perfeito para iniciantes.

Pense nele como seu “caderno inteligente”. 📓✨

✅ Passo 1: Baixar o VS Code
Acesse:
👉 https://code.visualstudio.com/
Clique em:
Download for Windows
✅ Passo 2: Instalar o VS Code
Abra o arquivo baixado
Aceite os termos
Clique em Próximo
⚙️ Atenção (IMPORTANTE)

Marque essas opções:

✔ Adicionar "Abrir com Code"
✔ Adicionar ao PATH (se aparecer)

💡 Isso facilita MUITO depois.

✅ Passo 3: Finalizar
Clique em Instalar
Depois em Concluir

✔ O VS Code vai abrir automaticamente

🧪 Testando

Se abriu uma tela de editor (clara ou escura), está funcionando ✔

🧩 Próximo passo (IMPORTANTE)

Agora precisamos ensinar o VS Code a entender Python.

🧩 Instalando o Suporte ao Python no VS Code

O VS Code não entende Python sozinho — vamos resolver isso.

✅ O que vamos instalar?

Extensão oficial:

👉 Python (da Microsoft)

Ela já traz:

Autocomplete 🤖
Execução de código ▶️
Debug 🐞
✅ Passo 1: Abrir extensões
Abra o VS Code
Clique no ícone de Extensões (🧩)
✅ Passo 2: Buscar

Digite:

Python
✅ Passo 3: Instalar
Escolha a extensão da Microsoft
Clique em Install
✅ Passo 4: Verificar
O botão Install some
Aparece ⚙️

✔ Instalado com sucesso

🔄 Reiniciar (se precisar)

Clique em Reload ou reinicie o VS Code

🧪 Teste rápido

Crie um arquivo:

teste.py

Digite:

print("Hello, World!")

Clique em ▶️ para rodar

✔ Se aparecer no terminal, deu certo!

👉 Criar seus primeiros algoritmos em Python
