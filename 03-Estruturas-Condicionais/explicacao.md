# 🚦 Módulo 03: Estruturas Condicionais em Python

Até agora, nossos códigos rodavam em uma linha reta, do começo ao fim. Mas e se o programa precisar tomar uma decisão? É para isso que servem as **Estruturas Condicionais**! Elas são as "bifurcações" ou "encruzilhadas" do nosso código.

Em Python, a forma como escrevemos essas decisões é incrivelmente limpa e legível, quase como ler um texto em inglês.

Para facilitar o seu estudo, dividimos este módulo em duas pastas.

### 📁 O que você vai encontrar aqui:

- **[If / Elif / Else](./if-else/)**: A estrutura "Se / Senão Se / Senão". É a ferramenta de decisão mais poderosa e usada na programação. Perfeita para checar regras flexíveis, intervalos e condições complexas (ex: _se a nota for maior que 7 E a falta menor que 5_).
- **[Match Case](./match-case/)**: A grande novidade do Python (a partir da versão 3.10)! É o equivalente ao "Switch" de outras linguagens. Uma estrutura super limpa e organizada, ideal para quando temos opções fixas e exatas (ex: _digite 1 para saldo, 2 para extrato, 3 para sair_).

Escolha uma das pastas acima para mergulhar na teoria e fazer os exercícios. Lembre-se de prestar muita atenção na **indentação** (os espaços antes do código), pois no Python ela é a regra mais importante! Bons estudos! 🚀
