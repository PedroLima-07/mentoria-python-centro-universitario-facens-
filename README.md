# 🎓 Mentoria Java - Centro Universitário Facens

Este repositório foi desenvolvido como material de apoio para a mentoria de programação em Java, com foco em lógica de programação para iniciantes.

---

## 🚀 Objetivo

Ajudar alunos a desenvolverem lógica de programação utilizando Java, passando pelos principais conceitos fundamentais:

- Variáveis
- Operadores
- Condicionais
- Estruturas de repetição

---

## 📚 Estrutura do Repositório

O conteúdo está organizado por etapas de aprendizado:

01-variaveis/
02-operadores/
03-condicionais/
04-repeticao/

Cada módulo contém:

- 📖 Explicações teóricas
- 💻 Exemplos práticos
- 🧪 Exercícios
- ✅ Resoluções

---

## 🧠 Ordem de Estudo Recomendada

1. Variáveis
2. Operadores
3. Condicionais (`if` e `switch`)
4. Repetição (`for`, `while`, `do-while`)

---

## 🔀 Condicionais

Estruturas que permitem que o programa tome decisões:

- `if / else` → decisões com lógica flexível
- `switch` → decisões baseadas em valores fixos

---

## 🔁 Estruturas de Repetição

Utilizadas para repetir ações:

- `for` → quando sabemos o número de repetições
- `while` → quando depende de uma condição
- `do-while` → executa pelo menos uma vez

---

## 🧪 Como estudar

1. Leia a explicação do conteúdo
2. Analise os exemplos
3. Tente resolver os exercícios
4. Compare com as resoluções
5. Pratique modificando os códigos

---

## 💡 Dica importante

> Aprender programação não é decorar, é praticar.

Tente sempre escrever os códigos por conta própria antes de olhar as respostas.
