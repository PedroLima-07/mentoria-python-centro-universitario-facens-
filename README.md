# 🐍 Trilha de Aprendizado: Lógica de Programação com Python

Boas-vindas ao nosso repositório de estudos! 🎉

Se você está dando os seus primeiros passos no mundo da programação, você está no lugar certo. Escolhemos o **Python** para essa jornada porque ele é uma linguagem poderosa, mas ao mesmo tempo muito amigável e legível. Quase como escrever em inglês!

Este repositório foi construído para ser o seu guia prático. Aqui não tem enrolação: é teoria direto ao ponto, muita prática e a resolução comentada para você conferir depois.

---

## 🛠️ Como usar este repositório?

Nossa trilha foi organizada em módulos evolutivos. O ideal é que você siga a ordem numérica para que a base fique bem sólida.

Em cada módulo (ou subpasta), você sempre encontrará a nossa "Tríade do Aprendizado":

1. 📖 **`explicacao.md`**: A teoria mastigada, com analogias e exemplos visuais.
2. 🏋️ **`exercicios.md`**: Desafios práticos para você criar seus próprios códigos `.py`.
3. ✅ **`resolucoes/`**: O gabarito comentado passo a passo para você comparar com a sua solução.

---

## 💻 O que eu preciso para começar?

Para rodar os códigos deste repositório, você vai precisar de um ambiente de desenvolvimento preparado:

1. **Python:** Faça o download e instale a versão mais recente [no site oficial](https://www.python.org/downloads/). _(Lembre-se de marcar a caixinha "Add Python to PATH" durante a instalação no Windows!)_.
2. **Editor de Código (IDE):** Recomendamos fortemente o [Visual Studio Code (VS Code)](https://code.visualstudio.com/), junto com a extensão oficial do Python.
