# 🎓 Mentoria Python - Centro Universitário Facens

Este repositório foi desenvolvido como material de apoio para a mentoria de programação em Python, com foco em lógica de programação para iniciantes.

## 🚀 Objetivo

Ajudar alunos a desenvolverem lógica de programação utilizando Python, passando pelos principais conceitos fundamentais:

- Variáveis
- Operadores
- Condicionais
- Estruturas de repetição
- Arrays (Listas)

## 📚 Estrutura do Repositório

O conteúdo está organizado por etapas de aprendizado:
`01-variaveis/` `02-operadores/` `03-condicionais/` `04-repeticao/` `05-arrays/`

Cada módulo contém:

- 📖 Explicações teóricas
- 💻 Exemplos práticos
- 🧪 Exercícios
- ✅ Resoluções

## 🧠 Ordem de Estudo Recomendada

1. Variáveis
2. Operadores
3. Condicionais (`if/elif` e `match-case`)
4. Repetição (`for`, `while`, simulando `do-while`)
5. Arrays (Listas)

## 🔀 Condicionais

Estruturas que permitem que o programa tome decisões:

- **if / elif / else** → decisões com lógica flexível.
- **match-case** → decisões baseadas em valores fixos (o "switch" do Python).

## 🔁 Estruturas de Repetição

Utilizadas para repetir ações:

- **for** → ideal para percorrer itens ou quando sabemos o número exato de repetições (usando `range()`).
- **while** → quando a repetição depende de uma condição verdadeira.
- **while True (com break)** → usado para menus e para garantir que o código execute pelo menos uma vez (simulando o `do-while`).

## 📦 Arrays (Listas)

Utilizados para agrupar dados:

- **Listas** → Em Python, os arrays são chamados de listas. Elas são dinâmicas, não possuem tamanho fixo e são perfeitas para serem percorridas com o laço `for`.

## 🧪 Como estudar

1. Leia a explicação do conteúdo.
2. Analise os exemplos.
3. Tente resolver os exercícios.
4. Compare com as resoluções.
5. Pratique modificando os códigos.

## 💡 Dica importante

Aprender programação não é decorar, é praticar.
Tente sempre escrever os códigos por conta própria antes de olhar as respostas!
